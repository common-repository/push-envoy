Push Envoy is a great plugin for engaging users on browsers and mobile via push notifications.

== Description ==

Get an opt-in rate of up to 25% easily with Push Envoy. Automatically send out push notification to your visitors on posting a content. With Push envoy, you can do the following:

1. Send  product offers or promote a specific content using the send instant message option
2. Convert your categories to segments so visitors can choose what category they want notification
3. Configure automatic sending of notification  to subscribers  once post is published
4. Manually choose to send out push notification once a post is edited
5. Clean dashboard for stats.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Push Envoy ->Settings screen to configure the plugin
4. Register a FREE Push Notification account. [Register](https://pushenvoy.com/app/login)
5. Provide Application Token.For detailed instructions please refer to the complete [Documentation](https://pushenvoy.com/docs/our-wordpress-plugin/how-to-setup-your-app-token/) page.
6. [Optional] Once your activation token has been successfully saved, you can choose to convert your category to push notification segments at any time.

For more information, read our user guide.[User Guide](https://pushenvoy.com/docs/our-wordpress-plugin/how-to-install-plugin/)



== Frequently Asked Questions ==

= What are push notifications? =

A push notification is a message that pops up on a web browser or mobile device. Push notifications look like SMS text messages and mobile/web alerts, but they only reach users who have installed your app or accepted push notifications on their browsers. 

= How to integrate to my website?=

* If you dont have an account, register  [Register] (https://pushenvoy.com/app/register)
* Once you confirm your email, log in to dashboard
* Go add new application Here
* Select quick desktop setup
* Select and configure your template then save
* Install this plugin and setup using your application token

= I have converted my WordPress to mobile app, can I use this plugin?=

Yes, you can, once you register to create an advanced application on your Push Envoy account. Ensure to generate your server keys for the mobile app type.
Please see our [Wiki](https://pushenvoy.com/documentation/)
